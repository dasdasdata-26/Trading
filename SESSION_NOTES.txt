HPM Proposal Generator Web - Session Notes
============================================
Date: March 3, 2026

PROJECT OVERVIEW
----------------
- Type: Flask + Vanilla JS web app for generating professional Word (.docx) proposals
- Location: D:\Dropbox\HPM AI\HPMProposal GeneratorWeb
- Run: python app.py or run.bat -> opens browser at http://localhost:5000
- No database -- JSON-based file storage in hpm_proposal/
- Desktop shortcut created: C:\Users\Owner\Desktop\HPM Proposal Generator.bat

KEY FILES
---------
- app.py -- Flask backend (routes, settings API, doc generation, draft management)
- document_generator.py -- Builds .docx proposals using python-docx
- hpm_proposal/settings.json -- App settings (company, user, footer, rate schedule)
- hpm_proposal/templates/ -- Proposal templates (JSON)
- hpm_proposal/drafts/ -- Saved draft proposals
- hpm_proposal/output/ -- Generated .docx/.pdf files
- templates/index.html -- Main proposal form (Jinja2)
- templates/settings.html -- Settings page (Jinja2 + inline JS)
- static/js/app.js -- Frontend logic
- static/css/style.css -- Styling

DEPENDENCIES
------------
- Flask, python-docx, beautifulsoup4 (see requirements.txt)
- Optional: LibreOffice for PDF export

HOSTING ON ANOTHER COMPUTER
----------------------------
1. Copy the entire project folder to the target machine
2. Install Python 3.7+ (check "Add Python to PATH" during install)
3. Open a terminal and run: pip install -r requirements.txt
4. Run: python app.py (or double-click run.bat on Windows)
5. Optional: Install LibreOffice for PDF export
