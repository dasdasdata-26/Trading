HPM Proposal Generator

Version 1.0 — Locally-hosted web application

Generates professional client proposals for HPM Systems, Inc. Runs entirely on your local computer — no internet connection required.

How to start: Run python app.py or double-click run.bat.

To add new templates: Create a JSON config file in hpm_proposal/templates/ and add its filename to templates.json.
